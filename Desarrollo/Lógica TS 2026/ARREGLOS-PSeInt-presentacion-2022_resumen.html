Introducción a los Arreglos

Un arreglo (también conocido como vector o array) es una estructura de datos que permite almacenar una colección de elementos homogéneos del mismo tipo en la memoria del ordenador.

Los arreglos se utilizan para resolver problemas que requieren la manipulación de grandes cantidades de datos, como el ejemplo de las facturas del año pasado.

Tipos de Estructuras de Datos

  • Estructuras Internas: residen en la memoria del ordenador y se utilizan para almacenar datos que se pueden acceder rápidamente.
  • Estructuras Externas: residen en un soporte externo, como un dispositivo magnético, y se utilizan para almacenar datos que se pueden acceder más lentamente.

Arreglos Unidimensionales

Un arreglo unidimensional es un conjunto de datos homogéneos que se pueden acceder mediante un índice.

El proceso de reservar una zona de la memoria para almacenar los datos del arreglo se llama dimensionar.

Un arreglo unidimensional se puede clasificar en los siguientes tipos:

  • Numérico: almacena números enteros o decimales.
  • Cadenas de caracteres: almacena cadenas de caracteres.
  • Booleanos: almacena valores booleanos (verdadero o falso).

Arreglos Bidimensionales

Un arreglo bidimensional es un conjunto de datos homogéneos que se pueden acceder mediante dos índices.

Un arreglo bidimensional se puede clasificar en los siguientes tipos:

  • Matrices: almacena datos en forma de tabla.
  • Matrices de matrices: almacena matrices en forma de tabla.

Acceso a los Elementos de un Arreglo

Los elementos de un arreglo se pueden acceder mediante un índice.

El índice se utiliza para especificar la posición del elemento en el arreglo.

El acceso a los elementos de un arreglo se puede realizar de manera secuencial o directa.

Recorridos por Fila y Columna

Un recorrido por fila es un recorrido que se realiza de manera secuencial, desde la primera fila hasta la última fila.

Un recorrido por columna es un recorrido que se realiza de manera secuencial, desde la primera columna hasta la última columna.
